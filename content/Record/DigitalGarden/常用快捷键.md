## ✍内容
> [!abstract] Windows
>Win + Ctrl + 左/右切换虚拟桌面
>Alt+F4，可以最快的方式关闭任何应用程序
>

> [!abstract] Ubuntu
>   Ctrl + Alt + T：打开终端
>  Ctrl + Shift + T：在同一个窗口新建终端标签页
>   Ctrl + Shift + W：关闭标签页
>     Ctrl + Shift + N：新建终端窗口
 Ctrl + Shift + Q：关闭终端窗口
